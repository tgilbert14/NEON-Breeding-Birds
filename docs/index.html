How the field guide reads

Every count starts with someone listening.

Birds are surveyed by timed point counts: an observer stands, listens, and watches. The field guide follows that same path — from a single count to a continent of communities, with the statistics shown at every step.

How it stays honest
Every view states its method, on the chart. Abundance is a detection index — birds per 6-minute point count, never a census, because a loud species and a quiet one at equal density give unequal counts. Ubiquity (% of points detected) is the less count-biased axis, though still a detection floor. Richness uses Chao2 and species accumulation on the point × year occasion as the incidence unit, and across sites is rarefied to a common number of point-counts, because raw richness just tracks effort. Detectability-by-distance is area-corrected to detections per hectare. The continental climate gradient is a space-for-time comparison of 46 contemporaneous places — correlational, confounded by biome and latitude, never read as one site warming; precipitation is shown only where NEON has a gauge, never imputed.
